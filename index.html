Welcome to Grokker Labs.

Grokker is a containerized, on premise, data science server. Grokker will connect to the datasource of your choice to get training data, clean the data, generate features, create predictive models, deploy models and generate queryable endpoints with only a few clicks through a simple dashboard. The model creation and deployment is distributed. App developers only have to connect a data source and generate endpoints for the automatically generated models. Also since it is on premise, there is no data transfer of any sort
