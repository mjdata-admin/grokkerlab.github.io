Welcome to Grokker Labs.

Grokker. It is - 1. a "platform for managing data science experiments and making the results available to other teams within an enterprise via prediction apps‚Äù. 2. a containerized, on-premise or hosted machine learning server which computes predictive models and provides API endpoints to access predictions; 3. a dashboard interface for data scientists used for creating and cleaning arbitrarily large and distributed data-sources, automatic feature engineering, creating and evaluating predictive models, string together predictive models, schedule modeling jobs and generate endpoints from multiple models 4. a dashboard interface for business users (web devs, marketing, sales, finance i.e. consumers of the results of data science experiments) used for creating predictive apps using the endpoints; apps for predicting conversion, recommending similar products, recommending similar pieces of texts, intelligent calendar notifications, auto emailers, etc. and iterate on the process themselves if they wanted to.
